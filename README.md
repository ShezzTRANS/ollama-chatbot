# AI Chatbot Project

## Overview
This is an AI chatbot project built with FastAPI and Ollama.

## Features
- FastAPI backend
- PostgreSQL/SQLite support
- Ollama LLM integration
- Simple frontend UI

## Setup

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Deployment
- Docker supported
- Deployable on Railway
